# 📱 Samsung Z Flip 5 外壳制作与建模项目(这个文档还没编辑完成，当前为模板)

**Samsung Galaxy Z Flip 5 Custom Case / CAD / Reverse Modeling Library**

本项目专注于 **三星 Z Flip 5 外壳的结构测量、建模、重绘与自定义设计**。
内容包含精确尺寸数据、外壳基础模型、可编辑的外壳模板、结构分析笔记以及适配 3D 打印的设计建议。
适用于 DIY 爱好者、3D 打印玩家、工业设计学生、结构工程学习者。

---

## ✨ 特性（Features）

* 📐 **精确结构尺寸**：包含外壳轮廓、铰链外部尺寸、副屏与摄像头位置
* 🧩 **可直接使用的 3D 模型**：提供 STL / STEP / F3D 格式基础外壳模型
* 🎨 **可二次创作的外壳模板**：用于贴图重绘、纹理设计、图案雕刻
* 🔧 **结构安全区分析**：包括不可侵入、不可压迫、必须保持间隙的区域
* 🖨️ **3D 打印优化**：不同材料（TPU / PLA）厚度建议与卡扣容差
* 📱 **适配手机所有状态**：考虑折叠 / 展开两种状态外壳的形变与间隙
* 🛠️ **可扩展设计**：支持加入支架、指环、挂绳孔等功能配件

## 📖 项目简介（Introduction）

Samsung Z Flip 5 采用铰链折叠结构，因此外壳设计不仅需要适配两段机身形态，还需要避免在折叠过程中造成碰撞或刮擦。本项目从实际测量出发，构建了适合二次开发的基础外壳模型，并提供结构分析与设计建议，可用于：

* 外壳重绘
* 创意个性外壳设计
* 3D 打印外壳制作
* 工业设计课程作业
* 手机配件二次开发

## 🔧 3D 打印设计建议（3D Printing Guide）

**推荐材料：**

* TPU（软壳）
* PLA（硬壳）
* ABS（高温适配）
* PETG（柔韧抗摔）

**建议厚度：**

| 部位 | TPU(mm) | PLA(mm) |
| -- | ------- | ------- |
| 边框 | 1.3–1.6 | 1.0–1.2 |
| 背板 | 1.2–1.5 | 0.9–1.1 |
| 卡扣 | 1.8–2.2 | 1.6–2.0 |

**容差建议：**

* 手机边角：+0.2 ～ +0.35mm
* 卡扣：-0.15 ～ -0.25mm
* 铰链区域：保留 1.0mm 以上安全间隙

## 🛠️ 使用方式（Usage）

你可以使用本库来：

1. 打开 3D 模型 → 修改尺寸或添加设计
2. 使用模板进行图案设计
3. 输出 STL → 导入切片软件（Cura / Bambu Studio）
4. 打印或加工你自己的外壳
5. 进行结构分析或二次开发

示例流程：

```
Fusion 360 → 导入 zflip5_case_base.f3d
           → 添加纹理 / 修改边框 / 增加装饰
           → 导出 STL
           → 切片打印
```
